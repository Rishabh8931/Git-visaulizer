# 🖥️ Terminal

Simulates a real Git terminal.

---

## ✨ Features

- Manual command input
- Command history (↑ ↓ keys)
- Persistent history (localStorage)
- Prompt display (user@machine)

---

## 🔧 Functions

### handleEnter()
- Reads user input
- Parses command
- Dispatches action

---

### Command History
- Stores previous commands
- Allows navigation

---

## 🎯 Purpose

To replicate real Git CLI experience.
