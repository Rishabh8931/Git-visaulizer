# 🌿 Commit Graph

Visual representation of Git commits.

---

## 🎨 Elements

- Circle → Commit
- Line → Parent relationship
- Colors → Branches
- Label → Commit ID
- HEAD pointer → Current position

---

## ⚙️ Logic

- Y-axis → commit order
- X-axis → branch separation
- Lines connect commits using parent

---

## ✨ Animations

- Commit pop-in
- Line drawing
- HEAD highlight

---

## 🎯 Purpose

Helps users visualize Git structure.
